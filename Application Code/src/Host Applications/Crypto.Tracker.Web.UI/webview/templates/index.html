Welcome

This web application is developed as part of a master's degree project. Its purpose is to analyze Ethereum wallets and transactions to detect phishing, money laundering, and other illicit activities using advanced machine learning models.

Before getting started, please ensure you have completed the following setup steps:

  • Added the Trained Models to the project's model directory.
  • Added your Etherscan API key to the project's settings.
  • Set up the database correctly with all necessary migrations applied.
  • Created a superuser account to manage the Django admin panel.

You can find more details about the setup process in the GitHub repository. The repository includes a README.md file that provides comprehensive instructions on how to configure and run this web application.
